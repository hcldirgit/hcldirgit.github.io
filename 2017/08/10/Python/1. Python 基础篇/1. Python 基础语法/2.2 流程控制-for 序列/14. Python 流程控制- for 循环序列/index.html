  • 条件循环也是如此，当条件为假，循环结束。

        Python for循环可以遍历任何序列的项目，如一个列表或者一个字符串。

range

  • range可以快速的生成一个序列
  • range（i，j,[,步进值]）
    • 如果所创建对象为整数，可以用range
    • i为初始值，不选默认为0
    • j为终止值，但不包括在范围内
    • 步进值默认为1
