常见apache php_admin_value php_admin_flag 设置

php_admin_value php_admin_flag php_value php_flag

        前面的两个参数用在httpd.conf 中，后面的两个用在.htaccess中

        至于 value 和 flag 的不同在于，前者用于设置值（如字符串）， 后后者只能是on off
