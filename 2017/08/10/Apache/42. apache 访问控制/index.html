apache 禁止trace或track防止xss攻击

        TRACE和TRACK是用来调试web服务器连接的HTTP方式。

        支持该方式的服务器存在跨站脚本漏洞，通常在描述各种浏览器缺陷的时候，把”Cross-Site-Tracing”简称为XST。

        攻击者可以利用此漏洞欺骗合法用户并得到他们的私人信息。

        禁用trace可以使用rewrite功能来实现

        或者还可以直接在apache的配置文件中配置相应参数
