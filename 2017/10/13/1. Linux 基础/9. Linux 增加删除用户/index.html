Linux 增加删除用户

  • Linux

1.增加用户

        命令：useradd
        语法：

        参数：

        -u：自定义UID

        -g：使其属于已经存在的某个组，后面可以分组 id ，也可以分组名

        -d：自定义用户的家目录

        -M：不建立家目录

        -s：自定义 shell

        ‘useradd’ 不加任何选项直接跟用户名，则会创建一个跟用户名同样名字的组。

        ‘-g’ 选项后面跟一个不存在的gid会报错，提示该组不存在。

        ‘-M’ 选项加上后则不建立用户家目录，但是在/etc/passwd文件中仍然有这个字段。但是使用 ls /home/user11 查看一下会提示该目录不存在。所以 ‘-M’ 选项的作用只是不创建那个目录。
