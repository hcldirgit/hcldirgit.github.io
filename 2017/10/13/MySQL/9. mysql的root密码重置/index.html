mysql的root密码重置

  • MySQL

mysql设置密码

        注意：如果mysql已经有密码，该命令要加上 -p

碰到mysql root密码忘记该如何做呢？

1.编辑mysql主配置文件my.cnf

        在 [mysql] 字段下添加参数

2.重启数据库服务

3.进入mysql数据库（此时数据库不用授权）

4.修改相应的用户密码

        使用mysql库

        更新一个表

5.修改/etc/my.cnf去掉skip-grant，重启mysql服务

        这样就可以用新密码登录了
