continue和break

        while在执行的时候，先看判断语句，只要为真，就执行语句，执行完语句以后，再执行判断语句，所以，一般都会在判断条件这里设置一个变量，在执行语句的时候对这个变量进行操作，一直执行到判断条件为假时，退出，执行后面的代码。如果刚开始判断就为假，则直接跳过while。
