  • Python

        sys模块提供了一系列有关Python运行环境的变量和函数。

常见用法

sys.argv

        可以用sys.argv获取当前正在执行的命令行参数的参数列表(list)。

变量 解释
sys.argv[0] 当前程序名
sys.argv[1] 第一个参数
sys.argv[0] 第二个参数
