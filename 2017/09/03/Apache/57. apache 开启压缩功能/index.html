apache 开启压缩功能

发表于 2017-09-03

        这里的压缩并不是对网站的图片压缩，而是对普通的静态文件，诸如html, js, css 等元素压缩。不要小看这个压缩功能，如果一个网站的请求量很大的话，这样可以节省海量带宽，在我国带宽资源非常昂贵，所以小小的一个压缩功能可以为企业节省不少的成本呢！下面就来看看如何配置它？

        首先，需要看一下我们的apache是否支持压缩功能。

        看看是否有mod_deflate
