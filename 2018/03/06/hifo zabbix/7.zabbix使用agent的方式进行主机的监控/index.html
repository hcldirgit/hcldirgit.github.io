zabbix使用agent的方式进行主机的监控

  • Zabbix

        步骤很简单

        Configuration–>Hosts–>右上方Create Host

  • Hostname要写真正的主机名，就是shell中运行hostname的主机名，或者叫FQDN名。
  • Visible name：显示的名称，自定义
  • 添加所属组：这里可以直接进行所属组的选择，如果没有想添加的组，还可以在New Group那里进行所属组的添加。
  • 监控方式：可以选择自己需要监控的方式进行监控，默认是agent的方式，当然也可以使用snmp方式进行监控。使用snmp方式进行监控的话就可以不安装agent客户端了。
  • 设备的描述：必须要写，根据约定写，比如什么时候添加的，添加用户，机器的作用。
  • Monitored by proxy：分布式用得到Monitored by proxy，等提到分布式监控的时候再提这个proxy。

        主机添加完了剩下的几项：

  • Template：模版
  • IPMI：智能平台管理接口（可以理解为远程管理卡）
  • Macros：宏定义
  • Host inventory：主机的资产信息，这个资产信息可以手填也可以自动获取
  • Encryption：加密相关

添加主机模版

        主机的模版可以进行搜索，同时一个模版还可以关联多个模版。不过最后要记住成功添加的关键要按上面箭头指的Add，而不是下面蓝色按钮的Add，否则添加是不成功的。

        添加完模版以后点击Add，主机添加模版完成，回到Host界面，等一会就会发现主机右侧的ZBX变成了绿色，说明已经成功开始监控了。比如图中的zabbix-agent
