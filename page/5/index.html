if 判断的几种用法

1.和文档相关的判断

        shell 脚本中 if 还经常判断关于档案属性，比如判断是普通文件还是目录，判断文件是否有读写执行权限等。常用的也就几个选项：

  • -e：判断文件或目录是否存在；
  • -d：判断是不是目录，并是否存在；
    -f：判断是否是普通文件，并存在；
  • -r：判断文档是否有读权限；
  • -w：判断文档是否有写权限；
  • -x：判断是否可执行。
